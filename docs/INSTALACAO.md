# Instalar o nvm

# Utilizar o node: 20.20.0
-> nvm install 20
-> nvm ls
-> nvm use 20.20.0

# Inicializar o projeto
-> npm init

# Instalar o next
-> npm install next@13.1.6

# Instlar o react
-> npm install react@18.2.0

# Instalar o react-dom
-> npm install react-dom@18.2.0

=> Observação: Para usar o .nvmrc com o comando nvm install ou nvm use precisa estar no tertminal do ubunto -> acessar a pasta do projeto e colocar a instrução no terminal

# No ubunto:
 ## Acessar diretório diferente:
    -> cd /mnt/d
    -> cd estudos
    -> cd CursoDev
    -> cd clone-tabnews

 ## Para Limpar o terminal:
